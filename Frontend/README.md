# SHIB Airdrop Frontend

## Features
- Wallet connection (MetaMask & WalletConnect)
- USDT balance checking
- Airdrop claiming
- Multi-level referral system
- Backend integration for user tracking
- Admin panel for management

## Setup
1. Open `index.html` in a web browser
2. Make sure backend is running on port 3001
3. Connect wallet and start using the airdrop

## Admin Access
- Click "Admin Panel" button in top-right corner
- Use admin key: `admin123` (change in backend .env file)
